このドキュメントは、YouTube.py3ライブラリの全108メソッドを含む包括的なリファレンスです。左のナビゲーションから各カテゴリを選択するか、上部の検索ボックスで特定のAPIを検索できます。

基本クラス

YouTubeAPI(api_key)

YouTube Data API v3のメインクラス。

パラメータ

api_key str YouTube Data API v3のAPIキー

使用例

from youtube_py3 import YouTubeAPI
import os

api_key = os.getenv('YOUTUBE_API_KEY')
yt = YouTubeAPI(api_key)

YouTubeAPIError

YouTube API関連のエラー例外クラス。

メソッド

  • is_quota_exceeded(): クォータ超過エラー判定
  • is_api_key_invalid(): APIキー無効エラー判定
  • is_not_found(): リソース未発見エラー判定
  • is_forbidden(): アクセス権限エラー判定
  • get_suggested_action(): 推奨アクション取得

使用例

try:
    video = yt.get_video_info("invalid_id")
except YouTubeAPIError as e:
    if e.is_quota_exceeded():
        print("APIクォータが制限を超えました")
    elif e.is_not_found():
        print("リソースが見つかりません")

search_videos(query, max_results=5, order="relevance")

動画を検索します。

パラメータ

query str 検索キーワード
max_results int 取得する最大結果数（デフォルト: 5）

import time

def continuous_monitor(channel_id, check_interval=3600):
    """継続的な監視（1時間間隔）"""
    last_video_id = None
    
    while True:
        try:
            result = yt.monitor_channel_for_new_videos(channel_id, last_video_id)
            
            if result['count'] > 0:
                print(f"🚨 新動画検出: {result['count']}本")
                # 通知処理（メール、Slack等）
                # send_notification(result)
                
                last_video_id = result['latest_video_id']
            
            time.sleep(check_interval)
            
        except YouTubeAPIError as e:
            print(f"❌ 監視エラー: {e}")
            time.sleep(300)  # 5分待機後再試行

# ユーザーがブラウザで認証後、認証コードを取得
code = input("認証コードを入力してください: ")
success = yt.complete_oauth_flow(flow, code, 'my_token.pickle')
if success:
    print("認証が完了しました")

get_oauth_info()

現在のOAuth認証情報を取得します。

戻り値

  • authenticated: 認証状態（bool）
  • scopes: 許可されたスコープのリスト
  • expires_at: トークンの有効期限
  • token_valid: トークンの有効性
  • has_refresh_token: リフレッシュトークンの有無

使用例

info = yt.get_oauth_info()
print(f"認証状態: {info['authenticated']}")
print(f"許可スコープ: {info['scopes']}")
print(f"トークン有効: {info['token_valid']}")
if info['expires_at']:
    print(f"有効期限: {info['expires_at']}")

check_oauth_scopes(required_scopes)

必要なスコープが許可されているかチェックします。

パラメータ

required_scopes list 必要なスコープリスト

戻り値

  • has_all_scopes: 全てのスコープが許可されているか
  • missing_scopes: 不足しているスコープのリスト
  • current_scopes: 現在許可されているスコープ

scope_check = yt.check_oauth_scopes(['upload', 'full'])

else:
